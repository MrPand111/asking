# Asking

一个问与答的app

# Document

- 当前版本1.4

- 实例请访问：http://39.106.58.159/

- 环境支持：Node

- 框架：vue.js + vantUI + uni-app

- 目录结构：

    ``` javascript
    // src目录下为开发者编码
    src
    |--- assets     // 存放静态文件
    |--- components // 封装的组件
    |--- network    // http请求+封装接口
    |--- router     // 配置路由
    |--- store      // 状态管理
    |--- style      // 全局样式
    |--- utils      // 工具函数
    |--- views      // 页面
    ```

- 注意事项：

    - 安装项目所需依赖：命令行执行npm install

    - 运行项目：命令行执行npm run serve
